GitHub Friend Finder 🐙🐱

Welcome! 🎉 Upload a JSON file containing GitHub user data, then select your state to find friends near you.

At a minimum, make sure your JSON file is structured like this:

{
  "users": [
    {
      "login": "username1",
      "avatar_url": "https://avatars.githubusercontent.com/u/123456?v=4",
      "state": "California"
    },
    {
      "login": "username2",
      "avatar_url": "https://avatars.githubusercontent.com/u/789012?v=4",
      "state": "New York"
    }
  ]
}

Make sure to include the state field for each user!

If you need help retrieving your organization’s users, check out GitHub's REST API documentation for listing organization members .

Here's a sample curl command you can use to fetch members:

curl -H "Authorization: token YOUR_GITHUB_TOKEN" \
https://api.github.com/orgs/YOUR_ORG/members

Steps:

  1. Click Choose File to upload your JSON file.
  2. Select your State from the dropdown menu.
  3. Click Find Friends to see up to 5 GitHub users from your state — with random cat avatars for fun!
